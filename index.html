How to print without auto-duplex:
1. Enable "test", click Download fronts PDF and print it: scale 100%, margins none, no "fit to page".
2. Take the sheet, flip it on the long edge (like a page), put it back in the tray.
3. Download & print backs PDF. Hold it to the light — does the back sit behind the front?
4. If left–right is swapped → switch "Flip" to short edge and repeat the test.
5. Disable "test" and print everything the same way: fronts first, then backs.
6. Multiple sheets: if the back lands on the wrong sheet (order reversed), enable "Backs last-to-first" — it prints backs from the last sheet, matching a flipped stack. Test with 2 sheets.
